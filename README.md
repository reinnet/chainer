# chainer
## Introduction
Chainer generates a list of service chains with the following configuration:

| Property          | Value        |
|:----------------- |:------------:|
| Length            | \[4, 7\)     |
| Per Instance Cost | 100\$        |
| Bandwidth         | 250 bps      |

Please note that customers pay the instance cost so it would be profit in the eye of datacenter owner.
